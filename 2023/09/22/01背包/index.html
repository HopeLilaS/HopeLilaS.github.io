01背包问题

2023/9/22 2023 C++ 动态规划 ACM 题解

问题简介

01背包问题是一道相当经典的动态规划\(（Dynamic Programming, DP）\)问题，动态规划是一种通过把原问题分解为相对简单的子问题的方式求解复杂问题的方法。解决动态规划问题的核心是推出当前问题的状态转移方程（跃迁方程）。

问题描述

有\(N\)件物品和一个容量是\(W\)的背包。每件物品只能使用一次。

第 \(i\) 件物品的体积是 \(w_i\)，价值是 \(v_i\) 。( \(weight\) , \(value\) )

求解将哪些物品装入背包，可使这些物品的总体积不超过背包容量，且总价值最大。输出最大价值。

输入格式

第一行两个整数，\(N\), \(W\)，用空格隔开，分别表示物品数量和背包容积。

接下来有\(N\)行，每行两个整数 \(w_i\), \(v_i\)，用空格隔开，分别表示第 \(i\) 件物品的体积和价值。

输出格式

输出一个整数，表示最大价值。

问题解决过程

创建 \(w[n]\) ，存放物体的体积。

创建 \(v[n]\) ，存放物体的价值。

创建一个状态矩阵 \(f\) ，用于动态记录当前背包的情况。（此处略去了第\(0\) 行和第 \(0\) 列）

\[ \left[\begin{matrix} f_{11} & f_{12} & \cdots & f_{1W} \\ f_{21} & f_{22} & \cdots & f_{2W} \\ \vdots & \vdots & \ddots & \vdots \\ f_{N1} & f_{N2} & \cdots & f_{NW} \\\end{matrix}\right] \]

将 \(f\) 的第 \(0\) 行与第 \(0\) 列初始化为 \(0\) ,表示当前背包中没有有价值的物品。

接下来遍历\(f\) 的每一行：

for (int i = 1; i <= N; i++)
{
    for (int j = W; j >= 0; j--)
    {
        if (j >= w[i])//如果背包装得下当前的物体
        {
            f[i][j] = max(f[i - 1][j], f[i - 1][j - w[i]] + v[i]);
        }
        else//如果背包装不下当前物体
        {
            f[i][j] = f[i - 1][j];
        }
    }
}

虽然01背包问题是一道经典的动态规划问题，但看上面的代码不难发现，其本质实际实际上还是枚举。

每一列f[1][1,2...W] 都存放着一种尽量放满的队列。

上述代码的含义是：

如果背包装不下物体

则此处可以装下最大价值物体和上个物体相同背包容量处相同，也就是不再向背包中装物品。

如果背包装得下当前的物品

则又分为两种情况：

在前一个物体中寻找一个情况,形成一个新的组合

使得weight[i]+weight[i-1]==W ，并且求出与前一个情况组合后物品总共的价值value1。

但是，是否存在这样一个情况，寻找上一种情况后，物体组合的价值甚至不如不放入这个物体，也就是说，之前有某种排列，使得利用相同的 j 空间，创造的 value2 大于 value1 ，这就是第二种情况：

不如维持前一种组合，不放入该物体

如此一来，就列举出了放满的全部情况，且 f[N][W] 的数值就是最优的结果。
